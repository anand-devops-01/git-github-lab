# Git Cheat Sheet

## Repository

```bash
git init
git clone <url>
git status
```

## Staging

```bash
git add .
git commit -m "message"
```

## Remote

```bash
git push
git pull
git fetch
```

## Branches

```bash
git branch
git switch
git merge
git rebase
```

## Advanced

```bash
git stash
git reset
git revert
git tag
git cherry-pick
```
