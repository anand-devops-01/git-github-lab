# Git Add

## What is `git add`?

The `git add` command adds changes from the working directory to the staging area.

## Syntax

```bash
git add <file-name>
```

Add all files:

```bash
git add .
```

## Example

```bash
git add README.md
```

## Real-world Use Case

Use `git add` to prepare files before creating a commit.

## Interview Question

**Q:** What is the staging area in Git?

**A:** It is the place where changes are prepared before they are committed.
