# Git Status

## What is `git status`?

The `git status` command shows the current state of the Git repository.

## Syntax

```bash
git status
```

## Example

```bash
git status
```

## Output

Displays:

* Modified files
* Untracked files
* Staged files
* Current branch

## Real-world Use Case

Run `git status` before every commit to check what will be committed.

## Interview Question

**Q:** Why do developers use `git status` frequently?

**A:** To check the current state of the working directory and staging area before committing changes.
