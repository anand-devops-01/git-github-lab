# Git Restore

## Introduction

Restores files to their previous state.

## Syntax

```bash
git restore <file>
```

## Example

```bash
git restore README.md
```

## Real World Use Case

Discard accidental local changes.

## Best Practices

Use carefully because uncommitted changes may be lost.

## Interview Question

**Q:** What does `git restore` do?

**A:** It restores a file from the last committed state or the staging area.
