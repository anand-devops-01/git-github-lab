# Git Push

## What is `git push`?

The `git push` command uploads your local commits to a remote repository.

## Syntax

```bash
git push
```

## Example

```bash
git push origin main
```

## Real-world Use Case

Push your latest code changes to GitHub after committing.

# Git Push

## Introduction

The `git push` command uploads your local commits to a remote repository such as GitHub.

## Syntax

```bash
git push <remote> <branch>
```

## Example

```bash
git push origin main
```

## Expected Output

Your latest commits are uploaded to the remote repository.

## Real World Use Case

Push completed work to GitHub so teammates can access it.

## Common Mistakes

- Pushing without committing changes.
- Pushing to the wrong branch.

## Best Practices

- Run `git status` before pushing.
- Pull the latest changes before pushing.

## Interview Question

**Q:** What does `git push` do?

**A:** It uploads local commits to a remote repository.
