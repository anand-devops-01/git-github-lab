# Git Commit

## What is `git commit`?

The `git commit` command saves staged changes into the Git repository history.

## Syntax

```bash
git commit -m "Your commit message"
```

## Example

```bash
git commit -m "Added Git documentation"
```

## Best Practice

Write short and meaningful commit messages.

✅ Good:

```text
Added Git status documentation
```

❌ Bad:

```text
update
```

## Interview Question

**Q:** Why are meaningful commit messages important?

**A:** They make project history easier to understand and maintain.
