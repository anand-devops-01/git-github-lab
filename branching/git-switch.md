# Git Switch

## Introduction

The `git switch` command is a modern way to switch branches.

## Syntax

```bash
git switch <branch-name>
```

## Example

```bash
git switch feature-login
```

## Real World Use Case

Move between branches quickly.

## Best Practices

Prefer `git switch` over `git checkout` when changing branches.

## Interview Question

**Q:** Why was `git switch` introduced?

**A:** To simplify branch switching.
