# Git Merge

## Introduction

The `git merge` command combines changes from one branch into another.

## Syntax

```bash
git merge <branch-name>
```

## Example

```bash
git merge feature-login
```

## Real World Use Case

Merge a completed feature into the main branch.

## Best Practices

Pull the latest changes before merging.

## Interview Question

**Q:** What is Git Merge?

**A:** It combines two branches into one.
