# Git Checkout

## Introduction

The `git checkout` command switches between branches or restores files.

## Syntax

```bash
git checkout <branch-name>
```

## Example

```bash
git checkout main
```

## Common Commands

```bash
git checkout main
git checkout feature-login
git checkout -b new-feature
```

## Real World Use Case

Switch to another branch while working on multiple features.

## Best Practices

Commit your changes before switching branches.

## Interview Question

**Q:** What is the purpose of `git checkout`?

**A:** It switches branches or restores files.
