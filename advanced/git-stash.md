# Git Stash

## Introduction

The `git stash` command temporarily saves your uncommitted changes without creating a commit.

## Syntax

```bash
git stash
```

## Example

```bash
git stash
git stash list
git stash pop
```

## Real World Use Case

Temporarily save your work before switching branches.

## Best Practices

- Use meaningful stash messages.
- Remove unused stashes.

## Interview Question

**Q:** What is `git stash` used for?

**A:** It temporarily saves uncommitted changes and cleans the working directory.
