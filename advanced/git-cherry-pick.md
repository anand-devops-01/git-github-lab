# Git Cherry-Pick

## Introduction

The `git cherry-pick` command applies a specific commit from one branch to another.

## Syntax

```bash
git cherry-pick <commit-id>
```

## Example

```bash
git cherry-pick a1b2c3d
```

## Real World Use Case

Move a bug fix from one branch to another without merging the entire branch.

## Best Practices

Cherry-pick only the commits you really need.

## Interview Question

**Q:** What is Git Cherry-Pick?

**A:** It copies a specific commit to the current branch.
